# huddle_cloud_keypair (Resource)

Manages an SSH keypair for authenticating to virtual machine instances.

## Example Usage

```hcl
resource "huddle_cloud_keypair" "example" {
  name       = "my-key"
  public_key = file("~/.ssh/id_rsa.pub")
}
```

## Schema

### Required

- `name` (String) Unique name for the keypair.
- `public_key` (String) OpenSSH-formatted public key to import. If omitted, a new key pair is generated by the API.

### Read-Only

- `api_id` (String) Internal API ID of the keypair.
- `created_at` (String) Timestamp when the keypair was created (RFC 3339).
- `fingerprint` (String) MD5 fingerprint of the public key.
- `id` (String) Unique identifier of the keypair.
- `updated_at` (String) Timestamp when the keypair was last updated (RFC 3339).

## Import

Import an existing keypair by its name:

```shell
terraform import huddle_cloud_keypair.example <keypair-name>
```
